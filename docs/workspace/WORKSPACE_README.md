# AntiGravity LightRAG Workspace

## Overview

This directory acts as the **Workspace** (or "Laboratory") for the LightRAG project agentic development. It contains the active source code repository alongside research materials, local experiments, and reference documentation.

## Directory Structure

* **`LightRAG/`**: The active **Git Repository** containing the source code.
* **`docs/`**: Workspace-level documentation and research notes.
* **`references/`**: PDFs and external reference materials (e.g., ACE framework, MemEvolve papers).
* **`langfuse/`**: Docker composition and configuration for the Langfuse evaluation environment.
* **`scripts/`**: Utility scripts unique to this local workspace environment.
