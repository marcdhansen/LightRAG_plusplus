# Reference Materials

This directory contains research papers, PDFs, and external reference materials related to the LightRAG project.

## Files

- **`ACE agentic context engineering summary.pdf`** - Summary of ACE framework for agentic context engineering
- **`Agentic Context Engineering: Evolving Contexts for Self-Improving Language Models.pdf`** - Full paper on agentic context engineering
- **`MemEvolve: Meta-Evolution of Agent Memory Systems.pdf`** - Research paper on agent memory system evolution
- **`final_report_BCBC.pdf`** - Final report from BCBC project
- **`MEMGRAPH_VECTOR_RESEARCH.md`** - Research on Memgraph vector capabilities and integration
- **`WEBUI_TROUBLESHOOTING.md`** - Web UI troubleshooting guide (duplicate in workspace for easy access)

## Usage

These materials provide research context and technical background for LightRAG development. Consult them when planning new features or researching approaches.
