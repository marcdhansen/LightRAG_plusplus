# Troubleshooting Documentation

This directory contains troubleshooting guides and solutions for common issues encountered by agents working on the LightRAG project.

## Files

- **beads_sync_issues.md** - Comprehensive guide to Beads (bd) synchronization problems and solutions

## Purpose

This troubleshooting documentation serves as a centralized knowledge base for:

1. **Tool Issues** - Problems with development tools and dependencies
2. **Sync Problems** - Git and Beads synchronization conflicts
3. **Environment Setup** - Common configuration and setup issues
4. **Service Dependencies** - Issues with external services like Automem and Langfuse

## Usage

When encountering issues:

1. Check this directory first for known solutions
2. Follow the documented troubleshooting steps
3. Update documentation when discovering new issues or solutions
4. Share learnings with other agents via `/reflect`
